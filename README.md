洇岚 是一个基于 Mirai (mirai-ts) 服务于 Furry 文化群聊的 QQ Bot。

## 关于 “直播助手版”

**直播助手版** 有别于 **经典版**， 主要面向于希望拥有自己的**直播推送**同时包含一些小的娱乐功能的 Bot 的主播，用于服务粉丝群，提升活跃度及粘性。

其他版本  [经典版 (yinlan-classic)](https://github.com/colour93/yinlan-classic)
